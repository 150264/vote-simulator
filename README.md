# Simulatore di voto

Web application per la simulazione e il confronto di diversi metodi di voto
e funzioni di scelta collettiva.

## FunzionalitÃ 
- Inserimento di alternative e votanti
- Preferenze espresse come ranking (A > B > C)
- Confronto tra:
  - Maggioranza assoluta e ballottaggio
  - Condorcet e metodo di Schulze
  - IRV (voto alternativo)
  - Borda
  - Coombs
